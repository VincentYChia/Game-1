## Implementation Strategy

### Auto-Path Generation Pattern

Instead of manually adding `iconPath` to every JSON, use convention:
```
iconPath = "{category}/{item_id}.png"
```

Where:
- **category** is determined from item type:
  - Materials → `"materials"`
  - Equipment weapons → `"weapons"`
  - Equipment armor → `"armor"`
  - Equipment tools → `"tools"`
  - Equipment accessories → `"accessories"`
  - Stations → `"stations"`
  - Devices → `"devices"`
  - Consumables → `"consumables"`
  - Enemies → `"enemies"`
  - Resources → `"resources"`

- **item_id** comes from:
  - Items: `materialId` or `itemId`
  - Enemies: `enemyId`
  - Resources: `resource_type` enum value + `"_node"` suffix for nodes

### Example Auto-Path Code

```python
def get_icon_path(item_def):
    \"\"\"Auto-generate icon path from item definition\"\"\"
    # Determine category
    if hasattr(item_def, 'material_id'):
        if item_def.category == 'consumable':
            category = 'consumables'
        elif item_def.category == 'device':
            category = 'devices'
        elif item_def.category == 'station':
            category = 'stations'
        else:
            category = 'materials'
        item_id = item_def.material_id
    elif hasattr(item_def, 'item_id'):
        # Equipment
        if item_def.slot in ['mainHand', 'offHand'] and item_def.damage != (0, 0):
            category = 'weapons'
        elif item_def.slot in ['helmet', 'chestplate', 'leggings', 'boots', 'gauntlets']:
            category = 'armor'
        elif item_def.slot in ['tool', 'axe', 'pickaxe']:
            category = 'tools'
        else:
            category = 'accessories'
        item_id = item_def.item_id

    return f"{category}/{item_id}.png"
```
